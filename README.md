## 📌 Deskripsi

**MoneyTrack** adalah aplikasi manajemen keuangan pribadi berbasis web yang dibangun menggunakan PHP dan MySQL.
Aplikasi ini memungkinkan pengguna untuk mencatat pemasukan dan pengeluaran, mengelola transaksi, serta menganalisis data keuangan melalui grafik interaktif.

---

## ✨ Fitur Utama

* 💰 Mencatat pemasukan & pengeluaran
* 📊 Dashboard ringkasan keuangan
* 📈 Grafik interaktif (Chart.js)
* 🗂️ Manajemen transaksi (CRUD)
* 🔍 Pencarian & filter transaksi
* 📅 Laporan keuangan bulanan
* 🔐 Sistem login & registrasi pengguna

## ⚙️ Cara Menjalankan

1. Clone repository:

   ```bash
   git clone https://github.com/nadhifshoeemagoldist-wq/MoneyTrack.git
   ```

2. Pindahkan ke folder:

   ```
   htdocs/
   ```

3. Buat database:

   ```
   moneytrack
   ```

4. Import file SQL melalui phpMyAdmin

5. Jalankan di browser:

   ```
   http://localhost/MoneyTrack
   ```
